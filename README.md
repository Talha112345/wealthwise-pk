# WealthWise.pk

Pakistan's Smart Investment Discovery Platform — built with React.

## Features
- Browse 40+ financial products across savings, investment, insurance and crypto
- 6-filter product search with AND logic
- Personalized recommendations based on your risk profile
- Portfolio builder with weighted return calculations
- Product comparison side by side
- Return projection calculator

## Tech Stack
- React (Create React App)
- React Router v6
- Context API (Portfolio, UserProfile, Compare)
- DummyJSON API
- Plain CSS with CSS variables

## Getting Started

```bash
npm install
npm start
```

Open http://localhost:3000

## Financial Logic
- Weighted Return = sum((allocation/total) × expectedReturn)
- Recommendations filter by risk, horizon, liquidity and budget simultaneously
- Risk distribution shows portfolio breakdown by risk category
